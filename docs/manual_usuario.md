# Manual de usuario del dashboard

## Acceso

Con la aplicación en ejecución, abrir `http://localhost:8501`.

## Vista ejecutiva

Presenta KPIs, perfiles ocupacionales con mayor conversión, evolución mensual y contexto macroeconómico. Se utiliza para decidir dónde enfocar las campañas.

## Vista operativa

Permite filtrar clientes por ocupación, segmento y resultado. Incluye análisis por canal, distribución de duración de llamadas y descarga de muestras filtradas.

## Vista técnica

Muestra métricas de calidad, historial de ejecuciones ETL, segmentos K-Means, representación PCA y matriz de correlación.

## Consideración importante

Las visualizaciones apoyan decisiones, pero no establecen causalidad. El dashboard debe interpretarse junto con el criterio comercial y técnico del equipo.
